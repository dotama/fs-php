fs.php - Simple File API

fs.php is a minimal http api to up/download files in a folder like structure. A very minimal of S3 if you want.

Endpoint

You can find the endpoint at . Note that you need a token for requests except for files that are explicitly made public readable.

Examples: Open API.md (public-read) - List files at the root - List all files in /artifacts/
